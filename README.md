# 🩺 Deep Learning for Type 2 Diabetes Risk Assessment

### Validation on the DiaBD-A Regional Dataset (Bangladesh)

This repository contains the implementation, data analysis, and predictive model developed as part of the research chapter "Deep Learning for Highly Accurate Type 2 Diabetes Risk Assessment: Validation on the DiaBD-A Regional Dataset," accepted for publication in an Elsevier edited volume on Artificial Intelligence and Computational Models in Metabolic Health.

The project presents a deep learning-based approach for early Type 2 Diabetes Mellitus (T2DM) risk assessment using the DiaBD-A regional clinical dataset. By leveraging a Sequential Neural Network and comprehensive data preprocessing techniques, the model aims to support early diagnosis and improve healthcare decision-making.

---

# 📌 Problem Statement

Traditional diabetes screening methods often rely on laboratory tests that can be costly, time-consuming, or inaccessible in resource-constrained environments. Additionally, many predictive models are developed using non-regional datasets, limiting their effectiveness for South Asian populations.

This project addresses these challenges by developing a region-specific deep learning model trained on the DiaBD-A dataset to improve early detection of Type 2 Diabetes.

---

# 📝 Project Overview

The proposed framework utilizes a Sequential Neural Network trained on the DiaBD-A dataset to identify complex relationships among clinical and physiological features associated with diabetes risk.

The project combines data preprocessing, feature scaling, deep learning, and web deployment to demonstrate an end-to-end healthcare AI solution.

---

# 🚀 Key Features & Results

- **Prediction Accuracy:** Achieved approximately **93.67% classification accuracy** on the testing dataset.
- **Clinical Features:** Utilizes 14 clinical attributes including BMI, blood pressure, pulse rate, age, family history, and lifestyle-related indicators.
- **Data Preprocessing:** Mean imputation, normalization, and feature scaling using StandardScaler.
- **Comparative Analysis:** Performance evaluated against traditional machine learning models including Logistic Regression and RFE-optimized models.
- **Deployment:** Integrated with a lightweight HTML interface for cross-platform accessibility.

# 📊 Model Architecture

The predictive model is implemented using a Sequential Neural Network consisting of:

- **Input Layer:** Standardized clinical features
- **Hidden Layer 1:** 32 Neurons (ReLU)
- **Hidden Layer 2:** 16 Neurons (ReLU)
- **Output Layer:** 1 Neuron (Sigmoid Activation)

The architecture is designed for binary classification to predict diabetes risk.
